# 产品截图

本目录用于存放 README 与文档中的展示截图，**不会**打进网站或管理后台构建产物。

## 目录说明

| 子目录 | 建议内容 |
|--------|----------|
| `admin/` | 管理后台：登录、文章、AI 选题、站点配置等 |
| `website/` | 站点前台：首页、列表页、文章详情等 |
| `deploy/` | 部署相关（可选） |

## 在 README 中引用

```markdown
![管理后台](docs/screenshots/admin/example.png)
```

建议 PNG/WebP，单张控制在约 500KB 以内，文件名使用英文短横线。
